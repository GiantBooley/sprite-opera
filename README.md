# Installation
Step 1: `git clone https://github.com/GiantBooley/fast-sprite-tools.git`\
Step 2: `cd fast-sprite-tools`\
Step 3: `mkdir build && cd build`\
Step 4: `cmake ..`\
Step 5: `make`\
